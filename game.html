How to Play

The Player will be given a random number at the start of the game.

There are four crystals below, each with "an assigned value that is hidden from the player. By clicking on a crystal, the assigned value to the Player's total score.

The Player wins the game by matching their total score to the random number. The game is lost if the Player's total score exceeds the random number.

Each time the game starts, the random number and values assigned to the crystals will change.
